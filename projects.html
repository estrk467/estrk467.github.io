Journey's

Journey's is an an android app that track a user's walking or running route. We used Google Maps API to play around with the tracking and mapping the route. The user is also able to save their journeys and view them later. The app is a way for users to have a visual representation of their route so they can be more motivated to be more active.

Check it out!

Languages and Tools: Java, Android Studio, JetBrains, Google Realtime Database, Firebase Authentication Google Play Store API, Google Maps API
